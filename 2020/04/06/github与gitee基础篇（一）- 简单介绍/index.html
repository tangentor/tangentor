github简介

GitHub是一个面向开源及私有软件项目的托管平台，因为只支持git 作为唯一的版本库格式进行托管，故名GitHub。

GitHub于2008年4月10日正式上线，除了Git代码仓库托管及基本的 Web管理界面以外，还提供了订阅、讨论组、文本渲染、在线文件编辑器、协作图谱（报表）、代码片段分享（Gist）等功能。目前，其注册用户已经超过350万，托管版本数量也是非常之多，其中不乏知名开源项目 Ruby on Rails、jQuery、python 等。

2018年6月4日，微软宣布，通过75亿美元的股票交易收购代码托管平台GitHub。【来源：百度百科】

git简介

git是一个分布式版本控制软件，最初由林纳斯·托瓦兹创作，于2005年以GPL发布。最初目的是为更好地管理Linux内核开发而设计。应注意的是，这与GNU Interactive Tools不同。 git最初的开发动力来自于BitKeeper和Monotone。

在这里强烈推荐廖雪峰老师的git教程

github中项目的快速寻找

在搜索的时候加入条件限制：

前缀 作用
in:name 对项目名进行条件匹配
in:readme 对readme文件进行条件匹配
in:description 对项目介绍进行条件匹配
stars:> 项目的stars数目限制
forks:> 项目的forks数目进行限制
language: 项目使用的编程语言
pushed:>YYYY-MM-DD 项目最晚更新时间
